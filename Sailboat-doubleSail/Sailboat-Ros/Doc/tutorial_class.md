
2 远程登入(terminal1)
```
ssh sjtu151@192.168.1.51 
password sjtu
```

3 编译(terminal1)
```
cd ~sailboat_ws/src/
tar -zxvf your_function.tar.gz

```

4 启动船上程序(terminal1)
```

./start_launch_onboat.shp
```

5 启动岸上程序(terminal2)
```
cd ~/sailboat_ws/src/Sailboat-Ros/Sh/onshore

```
